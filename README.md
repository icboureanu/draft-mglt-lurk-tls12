## kramdown-rfc2629

xml/txt draft is generated using
[kramdown-rfc2629](https://github.com/cabo/kramdown-rfc2629). 

### installing kramdown-rfc2629

```
gem install kramdown-rfc2629
```


### generating txt / xml draft

```
kdrf mydraft.mkd
```
